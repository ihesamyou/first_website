API ارزها

در این سایت چهار api تحت وب برای کار با ارزها وجود دارد (اعداد حتما باید به صورت انگلیسی وارد شوند):

  1. برای گرفتن آخرین نرخ ارزهای مختلف بر حسب ریال و بر اساس داده های سایت www.tgju.org میتوانید از آدرس "www.ihosseinu.info/api/currency" استفاده کنید. مثال:
    curl -L https://www.ihosseinu.info/api/currency
    >>> {"status":"success","data":[{"name":"لیر ترکیه", "price":18200},
    ...,
    {"name":"سوم قرقیزستان", "price":3540}]}
  2. برای گرفتن آخرین نرخ یک ارز خاص میتوانید از آدرس "www.ihosseinu.info/api/currency/ارز" استفاده کنید. مثال:
    curl -L https://www.ihosseinu.info/api/currency/یورو
    >>> { "status": "success", "data": { "name": "یورو", "price": 307960 } }
  3. برای تبدیل ریال به دیگر ارزها میتوانید از آدرس "www.ihosseinu.info/api/rial_converter/ارز/مقدار" استفاده کنید. مثال:
    curl -L https://www.ihosseinu.info/api/rial_converter/دلار/30000
    >>> { "status": "success", "data": 1.0390959864917522 }
  4. برای تبدیل دیگر ارزها به ریال میتوانید از آدرس "www.ihosseinu.info/api/currency_converter/ارز/مقدار" استفاده کنید. مثال:
    curl -L https://www.ihosseinu.info/api/currency_converter/لیر ترکیه/5600
    >>> { "status": "success", "data": 101920000 }
